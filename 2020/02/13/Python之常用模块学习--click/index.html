Python之常用模块学习–click

Click是一个Python软件包，用于以可组合的方式创建漂亮的命令行界面，所需的代码更少。 这是“Command Line Interface Creation Kit”的缩写。 它是高度可配置的，但具有开箱即用的明智默认设置。

平时我们用Python写命令行工具，一般会用到argparse。那click就是argparse的升级版。那么我们看看click是怎么使用的。
